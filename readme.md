
# project description

This project uses genetic algorithm to optimize a set of inputs
to a python function that maximize coverage, in order to try 
find crashes and other unexpected behavior

# installation

## using conda
run the following code to install the project using conda

```shell
conda create --name coverage python=3.14
conda activate coverage
pip3 install -r requirements.txt

# optional for packages used only for development
pip3 install -r dev_requirements.txt
```
